PROJECT OVERVIEW

The Exam Management System is a Spring Boot-based application tailored to simplify and automate the process of managing student exams along with hall-wise seating arrangements.

User Roles

  • Admin: Schedule exams, manage students, and oversee hall allocations
  • Student: View exam schedules and seating arrangements

Key Features

  • Automated hall and seat allocation
  • Role-based access: Admin and Student
  • Modern UI with intuitive dashboard
